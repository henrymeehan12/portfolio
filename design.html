Design & Engineering

Industrial design background, product engineering by trade. These are the projects where thinking and making happened in the same room.

Project 01

Guitar Building

Three electric guitars built from raw wood blanks. Figured maple tops, mahogany bodies, bird inlay fretboards, and custom airbrush finishes mixed from scratch. The process: sketch on paper → body design in Illustrator → glue-up → shaping → neck fitting → 60+ color airbrush system.

Project 02

Color Theory & Visual Design

Foundation coursework from industrial design school. Systematic color progressions built from hand-cut paper and mixed gouache. Every color in the composition labeled, mixed to a specific value, placed intentionally. Copic marker product illustrations done to the same level of rigor.

Project 03 — In Progress

NEXFORM

An autonomous engineering system that reads technical manufacturing drawings, extracts geometry and dimensions using computer vision, and builds 3D CAD models without human input.

The pipeline runs from raw PDF blueprints → OCR + YOLO detection → geometry extraction → SolidWorks/Inventor COM automation → DXF output ready for laser cutting and press brake programming. Built on Python, trained on thousands of real shop drawings.

This is the intersection of product engineering knowledge and software — ten years of understanding how parts get made, automated.

SMART STEEL

PDF → geometry extraction. OCR + computer vision pipeline reading drawings the way an engineer would.

FORGE

Geometry → 3D model. COM automation layer that drives CAD software via extracted parameters.

EZDIM

3D model → validated DXF. Dimensioning and blueprint verification before the part goes to the shop floor.
